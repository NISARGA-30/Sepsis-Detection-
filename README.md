# Sepsis Detection Using Deep Learning

This project uses Transfer Learning with ResNet50 to detect bacterial infections from blood smear microscopy images.

## Features
- Blood smear image classification
- ResNet50 Transfer Learning
- Binary classification
- Infection / No Infection prediction

## Technologies
- Python
- TensorFlow
- Keras
- OpenCV
- Google Colab

## Results
- Accuracy: 95%+
- Precision: High
- Recall: High
- F1 Score: High
